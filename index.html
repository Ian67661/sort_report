學習心得

在這次開發排序演算法視覺化工具的過程中，我深刻體會到了「資料結構與演算法」結合「視覺化技術」的威力。原本枯燥的 O(n log n) 公式，透過動畫展示其「分而治之」的拆解過程後，變得非常直觀易懂。

我學會了如何使用 JavaScript 捕捉演算法內部的每一回合狀態（Step-by-Step generation），並利用 CSS 玻璃擬態與霓虹效果美化 UI。最重要的是，我理解到沒有最完美的排序法，只有「最適合目前場景」的排序法：例如插入排序在小規模或近乎排序完成的資料中表現極佳，而快速排序則是處理海量資料的首選。這段旅程不僅增進了我的程式技巧，也培養了我解決問題的思維。
