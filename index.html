氣泡排序法 (Bubble Sort)

氣泡排序是一種簡單的排序演算法，用於將一系列元素依照大小順序排列。它的原理是透過多次比較相鄰的元素，若順序錯誤則交換，逐步將最大或最小的元素「冒泡」到序列的一端。

基本想法

  1. 從陣列的開始位置依序比較每對相鄰的元素。
  2. 如果前一個元素比後一個元素大（需遞增排序），就交換它們的位置。
  3. 如此一輪過去後，最大的元素會出現在陣列的最後面。
  4. 重複以上步驟，每一輪比較的範圍逐漸縮小，直到全部元素排列完成。

示例範例 (JavaScript 版本)

function bubbleSort(arr) {
    let n = arr.length;
    for (let i = 0; i < n - 1; i++) {
        for (let j = 0; j < n - 1 - i; j++) {
            if (arr[j] > arr[j + 1]) {
                // 交換
                let temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
            }
        }
    }
    return arr;
}

let nums = [5, 2, 9, 1, 5, 6];
console.log(bubbleSort(nums)); // [1, 2, 5, 5, 6, 9]

時間複雜度

氣泡排序的平均與最壞情況時間複雜度皆為 O(n2)，最佳情況（已排列的資料）為 O(n)。

優點與缺點

  • 實作簡單，容易理解。
  • 效率較低，不適合大型資料集合。
  • 穩定（相等元素的相對順序不會改變）。

互動式示範

輸入一組以逗號分隔的數字，點選「排序」按鈕即可觀看氣泡排序依步驟運作的動畫。

結論

雖然氣泡排序效率並不高，但因其直觀的機制，常用於教學或當作更複雜演算法前的入門示例。
